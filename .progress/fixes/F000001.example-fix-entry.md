# Fix — Example progress entry

## Summary

This file demonstrates a **Fix** Dot Progress entry: a bug fix or regression repair recorded as its own append-only audit event. The filename uses the `F` prefix with a six-digit global counter; frontmatter `milestone` is set to `Fix`.

## Changes

- Added `.progress/fixes/F000001.example-fix-entry.md` — this example

## Decisions

- Fix entries use `F{nnnnnn}.{descriptor}.md` — one six-digit number, separate counter from `G` and `M`.
- Frontmatter `milestone` is the literal string `Fix`.
- Fixes get new files even when reversing a prior change — never edit an earlier progress entry.
