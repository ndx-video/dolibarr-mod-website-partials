# P0 — Scaffold and enable modWebsitePartials

Scaffolded the slim Dolibarr 22 external module `websitepartials` from the ModuleBuilder template (descriptor only — no CRUD objects, SQL, menus, or public/REST handlers). Enabled and verified on the local module-dev stack at `https://partials.gandalf.lan`.

- Added `htdocs/custom/websitepartials/core/modules/modWebsitePartials.class.php` — descriptor (`numero` 500000, `family` interface, `version` development, `depends` modWebsite)
