# P1 — Public partials router

## Summary

Shipped unauthenticated published islands as `.html` and `.json` via `public/partial.php`, with Caddy rewrite for ROADMAP pretty URLs on `partials.gandalf.lan`.

## Changes

- Added `public/partial.php` front-controller (NOLOGIN bootstrap, IP allowlist, publish gate, Cache-Control)
- Added `websitepartials_page_to_public_array()` for the 4-field public JSON contract
- Updated `dev/caddy/partials.gandalf.lan.caddy` and live dock Caddyfile: rewrite `/public/partials/*` → `partial.php{uri}`
- Documented public surface in module README / ChangeLog; ticked P1 Done-when in ROADMAP

## Decisions

- Stock Dolibarr Apache has no `mod_rewrite`; PATH_INFO + Caddy rewrite is the canonical path
- Direct fallback URL remains `/public/partial.php/partials/{ref}/{slug}.{ext}`
- Drafts and missing pages both return `404`; malformed paths return `400`
- Page PHP is never executed — raw `content` only
