# General — Example progress entry

## Summary

This file demonstrates a **General** Dot Progress entry: work that is meaningful to record but not tied to a roadmap milestone. The filename uses the `G` prefix with a six-digit global counter; frontmatter `milestone` is set to `General` (not a roadmap label).
