# Adopt Dot Progress convention

## Summary

Adopted the [DotProgress](https://github.com/ndx-video/DotProgress) append-only progress-log convention for this repo (same layout as cloudflare-worker-braypark). Planning stays in `ROADMAP.md` (P0–P4); history is recorded by appending new entries under `.progress/`.

- Added `.progress/README.md` — progress log spec
- Added `.progress/milestone/`, `.progress/general/`, `.progress/fixes/` with example entries
- Added `AGENTS.md` and `CLAUDE.md` — agent orientation
- Added `.claude/rules/dotprogress.md` and `.claude/skills/dotprogress/SKILL.md`
- Expanded `README.md` for humans

## Decisions

- Map ROADMAP **P0–P4** to Dot Progress filename prefixes **M000–M004** (documented in AGENTS.md).
- `.progress/` is tracked in git and not gitignored.
- Consumer (Astro) wiring history remains in cloudflare-worker-braypark `.progress/`.
